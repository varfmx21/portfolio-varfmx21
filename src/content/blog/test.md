## Introducción

Este es un post de ejemplo para demostrar cómo funciona el renderizado de **Markdown** en Astro.

## Características

Puedes usar todas las características de Markdown:

- Listas con viñetas
- **Texto en negrita**
- *Texto en cursiva*
- `Código inline`

### Bloques de código

```javascript
const saludo = "¡Hola desde Markdown!";
console.log(saludo);
```
